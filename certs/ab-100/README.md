# AB-100 — Agentic AI Business Solutions Architect

> Diseño de soluciones de IA agéntica enterprise sobre la plataforma Microsoft: Azure AI Foundry, Copilot Studio, Microsoft 365 Copilot, Power Platform.

📅 **Estado:** ✅ certificado (2026)
🔗 **Study guide:** [learn.microsoft.com](https://learn.microsoft.com/es-mx/credentials/certifications/resources/study-guides/ab-100)

---

## Rol del candidato

El AB-100 es el **arquitecto** del stack: define qué construir, qué servicios usar, cómo gobernar costos, seguridad y Responsible AI. No implementa el código (AI-103) ni opera el ciclo de vida (GH-600).

```
AB-100  →  Define qué construir (arquitectura, diseño, gobernanza)
AI-103  →  Construye el núcleo de IA (Python, RAG, agentes)
GH-600  →  Automatiza el ciclo de vida (CI/CD, Copilot agents, MCP)
```

## Temas centrales del examen

- **Diseño de soluciones agénticas** — agentes autónomos, task agents, orquestación multi-agente
- **Ecosistema Microsoft AI** — Azure AI Foundry, Copilot Studio, Microsoft 365 Copilot, Power Platform
- **Protocolos abiertos** — Model Context Protocol (MCP), Agent2Agent (A2A)
- **Arquitectura enterprise** — ALM para IA, estrategia de environments, soluciones cross-platform
- **Responsible AI** — gobernanza, compliance, defensa contra prompt injection, residencia de datos
- **Alineación con negocio** — análisis de ROI, frameworks de adopción de IA, Cloud Adoption Framework
