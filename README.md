## 🔗 Caso práctico transversal

Un mismo proyecto cubre tres certificaciones desde ángulos complementarios:

➡️ **[Bot de WhatsApp para club deportivo](./casos-practicos/bot-whatsapp/)** — Azure AI Foundry + GitHub Copilot agents + CI/CD, visto desde AB-100 (arquitectura), AI-103 (implementación) y GH-600 (operación).

```
AB-100  →  Define qué construir (arquitectura, diseño, gobernanza)
AI-103  →  Construye el núcleo de IA (Python, RAG, agentes)
GH-600  →  Automatiza el ciclo de vida (CI/CD, Copilot agents, MCP)
```
